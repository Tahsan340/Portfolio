StudyBuddy

StudyBuddy is a console-based Java productivity application designed to help students manage their studies more effectively. It combines task management, smart task prioritization, Pomodoro-based focus sessions, progress tracking, streak monitoring, motivation support, routine generation, and a chatbot assistant in a single system. The project is built using Object-Oriented Programming (OOP) principles and Java Serialization for persistent data storage, allowing users to save and continue their study progress across sessions.
